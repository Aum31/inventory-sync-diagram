Manufacturer Feeds Specification

Format: Comma-separated values (.csv) or Excel spreadsheets (.xlsx).

Protocols: Frequently retrieved securely via SFTP, secure HTTPS links, or supplier API endpoints.

Key Fields: Requires Manufacturer SKU (acting as the absolute source of truth), wholesale price, and current stock level.

Python ETL Validator Engine

Extraction: Python script connects to feed servers, downloads the data, and reads it using modular readers.

Transformation: Sanitizes data types (converts strings to floats, strips whitespace, handles missing headers).

Concurrency: Optimized execution using asynchronous request handling to map multiple manufacturers concurrently.

Validation Gate (Rules Engine)

Negative Price Block: Automatically halts publishing if price <= 0.00 to avoid critical retail mistakes.

Margin Shield: Blocks updates where the wholesale price rises above the current retail price (preventing selling at a loss).

Out-of-Stock Override: Flags instances of empty stock strings and defaults them to 0 rather than keeping the old numbers.

WooCommerce & Shopify REST APIs

Shopify Endpoint: PUT requests to `/admin/api/2023-10/inventory_levels/set.json` using API access tokens.

WooCommerce Endpoint: POST requests to `/wp-json/wc/v3/products/batch` using consumer key and secret credentials.

Rate-Limits: Shopify defaults to 4 req/sec. Implement sleep statements to throttle batch transactions and avoid 429 timeouts.
